# servers are staying running even servers are rebooted.
>>services help to configure to run automatically any software.

>>service httpd start >> start httpd

>>systemctl start httpd >>newer way to manage services

>>systemctl stop httpd >> stop httpd 
>>systemctl status httpd >> check status of httpd 
>>systemctl enable httpd > configure httpd to start at startup
>>systemctl disable httpd > configure httpd to not start at startup

/etc/systemd/system/myapp.service

Description=My app

[Service]
ExecStart=/usr/bin/python3 myapp.py

>>systemctl daemon-reload >>let system know new service being configured.
